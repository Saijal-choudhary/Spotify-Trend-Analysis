## 3Ô∏è‚É£ Data Source

The dataset is based on Spotify music streaming data containing structured information such as:

- Song Name  
- Artist Name  
- Album Type (Single / Album)  
- Release Date  
- Popularity Score  
- Duration (Minutes)  
- Explicit Content Indicator  
- Year & Month of Release  

The data is organized in tabular format, enabling relational modeling between songs, artists, and album categories. This structure allows aggregation at multiple levels including artist-wise, album-wise, monthly, and yearly analysis.
